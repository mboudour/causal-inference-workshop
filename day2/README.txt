Day 2 — Identification, Estimation, and LLM Measurement Error
=============================================================

Goal: Apply causal estimators to LLM-measured outcomes (sentiment scores)
and demonstrate how measurement error type (MCAR vs MNAR) affects causal conclusions.

Data
----
data/speeches_sample.csv            : 4000 congressional speeches (111th Congress)
data/speeches_with_sentiment.csv    : speeches + LLM sentiment score (generated by partA)
data/speeches_with_sentiment_R.csv  : same, generated by R version of partA
data/embeddings_cache.npy           : cached sentence embeddings (auto-generated)

Python Scripts (run in order from python_app/)
-----------------------------------------------
1. partA_sentiment.py         : Compute LLM sentiment scores; save speeches_with_sentiment.csv
2. partB_estimators.py        : Estimate ATE using Diff-in-Means, Regression, G-formula,
                                Matching, IPW, AIPW (with standard errors and 95% CIs)
3. partC_measurement_error.py : Demonstrate MCAR vs MNAR measurement error and resulting bias
4. compare_days.py            : Compare Day 1 (stance) vs Day 2 (sentiment) ATE estimates

R Scripts (run in order from r_app/)
--------------------------------------
1. partA_sentiment.R          : Load embeddings from Python cache; compute sentiment scores
2. partB_estimators.R         : Same estimators as Python Part B
3. partC_measurement_error.R  : Same MCAR/MNAR diagnostics as Python Part C

Requirements
------------
Python: see requirements.txt
R packages: readr, dplyr, MatchIt, reticulate
